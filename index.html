Projects

2025.06 - Present

On-device VLM-based Robotic Mission Execution

Developed a natural-language mission execution framework for a ROS2-based Traxxas TRX-4 UGV platform that converts human commands into executable robot behaviors. In this framework, an on-device VLM generates high-level plans as pseudo-function sequences such as move, pick, place, search, and return-to-base, while the robot grounds these plans using RTAB-Map-based visual SLAM, object detection, and structured world-state representations.

To reduce infeasible or hallucinated actions, designed an Actor-Guard-Critic validation loop: the Guard first filters rule-violating plans, and the Critic checks contextual consistency before execution or replanning. The validated plan is then connected to ROS2/Nav2-based motion execution on a UGV platform.

Currently extending the framework with AirSim-based perception and planning: applying YOLOE, NanoOWL, and Grounding DINO for object detection on AirSim data, using NVBlox for 3D reconstruction, and generating paths with the OMPL library.

2024.11 - Present

ERP42 Autonomous Driving and Parking

Worked on a real ERP42 autonomous vehicle platform as a member and later team leader of KAUVOY. For localization, implemented a Dual EKF architecture using ROS robot_localization: a local EKF estimates short-term relative motion from IMU and wheel encoder measurements, while a global EKF incorporates RTK-GPS to correct long-term drift and provide a stable global pose for autonomous driving.

The localization system was evaluated on outdoor vehicle tests, achieving ATE (RMS) of 0.073 m on a one-lap run and 0.146 m after three laps. Based on this pose estimate, developed an autonomous parking pipeline covering cone perception, parking-space decision, Hybrid A* path planning, and Pure Pursuit tracking. Currently leading FSM-based decision-making logic for urban driving scenarios.
