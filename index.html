Selected Projects

ASOL
Jun. 2025 - Present

On-device VLM-based Robotic Mission Execution

Developed a natural-language mission execution framework that converts user commands into executable robot behaviors on a Traxxas TRX-4 UGV platform. The system combines an on-device VLM planner, Actor-Guard-Critic validation, RTAB-Map-based visual SLAM, object detection, structured world states, and ROS2/Nav2 control.

VLM ROS2/Nav2 RTAB-Map Jetson Reference code
KAUVOY
Nov. 2024 - Present

ERP42 Autonomous Driving and Parking

Implemented Dual EKF localization using IMU, wheel encoder, and RTK-GPS measurements on a real ERP42 autonomous vehicle. Developed and tested an autonomous parking pipeline with cone perception, parking-space decision, Hybrid A* path planning, and Pure Pursuit tracking. Currently leading FSM-based decision-making logic for urban driving scenarios.

Dual EKF Hybrid A* Pure Pursuit FSM
